## Title:MobileUse: A GUI Agent with Hierarchical Reflection for Autonomous Mobile Operation

Authors:[Ning Li](https://arxiv.org/search/cs?searchtype=author&query=Li,+N), [Xiangmou Qu](https://arxiv.org/search/cs?searchtype=author&query=Qu,+X), [Jiamu Zhou](https://arxiv.org/search/cs?searchtype=author&query=Zhou,+J), [Jun Wang](https://arxiv.org/search/cs?searchtype=author&query=Wang,+J), [Muning Wen](https://arxiv.org/search/cs?searchtype=author&query=Wen,+M), [Kounianhua Du](https://arxiv.org/search/cs?searchtype=author&query=Du,+K), [Xingyu Lou](https://arxiv.org/search/cs?searchtype=author&query=Lou,+X), [Qiuying Peng](https://arxiv.org/search/cs?searchtype=author&query=Peng,+Q), [Jun Wang](https://arxiv.org/search/cs?searchtype=author&query=Wang,+J), [Weinan Zhang](https://arxiv.org/search/cs?searchtype=author&query=Zhang,+W)

[View PDF](https://arxiv.org/pdf/2507.16853) [HTML (experimental)](https://arxiv.org/html/2507.16853v1)

> Abstract:Recent advances in Multimodal Large Language Models (MLLMs) have enabled the development of mobile agents that can understand visual inputs and follow user instructions, unlocking new possibilities for automating complex tasks on mobile devices. However, applying these models to real-world mobile scenarios remains a significant challenge due to the long-horizon task execution, difficulty in error recovery, and the cold-start problem in unfamiliar environments. To address these challenges, we propose MobileUse, a GUI agent designed for robust and adaptive mobile task execution. To improve resilience in long-horizon tasks and dynamic environments, we introduce a hierarchical reflection architecture that enables the agent to self-monitor, detect, and recover from errors across multiple temporal scales-ranging from individual actions to overall task completion-while maintaining efficiency through a reflection-on-demand strategy. To tackle cold-start issues, we further introduce a proactive exploration module, which enriches the agent's understanding of the environment through self-planned exploration. Evaluations on AndroidWorld and AndroidLab benchmarks demonstrate that MobileUse establishes new state-of-the-art performance, achieving success rates of 62.9% and 44.2%, respectively. To facilitate real-world applications, we release an out-of-the-box toolkit for automated task execution on physical mobile devices, which is available at [this https URL](https://github.com/MadeAgents/mobile-use).

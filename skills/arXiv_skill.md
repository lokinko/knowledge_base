# arXiv skill

## 适用范围

用于处理 arXiv 相关资料的知识化整理，适用于：

- `sources/` 下 frontmatter 中 `status: "new"` 的 `.md`
- 来源为 arXiv 摘要页、论文页，或附带 arXiv `HTML` / `PDF` 链接的剪藏
- 原文含网页噪声、英文或中英混合内容

不适用于：

- 非 arXiv 来源的一般网页文章
- 已处理完成的文件
- 只要求逐句翻译、不要求知识重写的任务

## 目标

将原始剪藏整理为可长期复用的中文知识笔记，重点是：

- 去噪
- 结构化重写
- 提炼问题、方法、结论与边界

## 输出要求

- 输出到 `summary/`，文件名与原文件一致
- 使用中文，保留关键术语，避免逐句直译
- 避免口语化表达，采用学术文章的表达结构与书面格式
- 信息不足时明确受限，不补写未证实细节
- 文末追加：`> 来源：[[原文件名]]`

## 最小流程

1. 找出 `sources/` 中 `status: "new"` 的目标文件。
2. 读取原始 markdown，并优先补读 `source`、`HTML`、`PDF` 中可访问的原文。
3. 搜索和该内容相近的 3 个文档，可以缺少但不能多，协同一起分析。
4. 去除导航、广告、页脚、重复段落等网页噪声。
5. 提取核心信息：研究问题、方法机制、实验或证据、结论与局限。
6. 重写为中文知识笔记，写入 `summary/` 同名文件。
7. 将原文件状态改为 `processed`，避免重复处理。

## 质量判断

合格结果应满足：

- 能快速理解论文核心贡献
- 结构明显优于原始剪藏
- 没有将摘要页信息误写为全文结论
- 没有无依据扩写
- 不要有`作者认为`这样的表述，统一采用`论文`等更三方的语句

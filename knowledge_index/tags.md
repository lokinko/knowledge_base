# Tags

Updated: 2026-04-22

## Tag Vocabulary

- `Agent Memory`: agent 长期记忆、外部记忆、服务期学习、多会话记忆。
- `GUI Agent`: 手机、网页、OS 图形界面操作 agent。
- `Mobile Agent`: 移动端 GUI agent 和移动助手。
- `Long-Horizon`: 多步骤、长程任务、错误累积和轨迹稳定性。
- `Benchmark`: 评测数据集、评测环境、指标和 benchmark taxonomy。
- `Personalization`: 用户 profile、偏好、习惯、隐式意图。
- `Proactivity`: 主动帮助、主动询问、介入校准。
- `Consent`: 同意协商、拒绝后克制、授权边界。
- `Reflection`: 自反思、对比式反思、层级反思、错误恢复。
- `Function Calling`: API/function/tool 调用、参数填充、工具选择。
- `Continual Learning`: 服务期学习、参数更新、稳定性-可塑性。
- `Multi-Agent`: 多智能体协作、协议、路由、收益分配。
- `LLM Knowledge Base`: LLM 维护 markdown wiki、索引、lint 和查询产物。
